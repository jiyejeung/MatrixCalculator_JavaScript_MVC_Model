### 목표: 👨🏻‍💻

- MVC Model을 기반으로 코드를 작성한다.
- ES6를 기반으로 코드를 작성한다.(module, arrow function, class...)
- 반복문과 배열 고차함수 활용을 목표로 한다.
- git과 github 사용을 목표로 한다.
- 함수는 가능하면, 한 가지의 일만 하도록 설계한다.(15줄을 넘기지 않는다.)
- HTML은 PUG를 통하여, CSS는 SASS를 통하여 만든다.
